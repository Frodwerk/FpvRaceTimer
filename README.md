# FpvRaceTimer
🏁 FPV Pro Race Timer

Ein leichter, browserbasierter Renn-Timer für FPV-Drohnen, der direkt über die Handykamera funktioniert. Das System erkennt Durchflüge mittels Bildanalyse in Echtzeit und berechnet Rundenzeiten sowie das beste "Triple" (drei aufeinanderfolgende Runden).

🚀 Funktionen
Echtzeit-Durchflugerkennung: Nutzt die Kamera, um Bewegungen im Bild zu erfassen.
Automatisierter Countdown: Startet das Rennen mit einer 3-Sekunden-Ansage.
Sprachausgabe: Akustische Rückmeldung der Rundenzeiten und Rennereignisse.
Triple-Berechnung: Identifiziert automatisch die schnellste Sequenz von drei aufeinanderfolgenden Runden.
Daten-Export: Rundenzeiten können für die Analyse einfach kopiert werden.
🛠 Anleitung zur Nutzung
Öffne den Link zu dieser Seite in deinem mobilen Browser (vorzugsweise Chrome oder Safari).
Erlaube den Zugriff auf die Kamera, wenn du dazu aufgefordert wirst.
Platziere das Handy stabil auf einem Stativ oder einer Halterung, sodass das Durchflugtor genau im Bild ist.
Wähle die gewünschte Flugrichtung (L->R oder R->L).
Klicke auf "START", um den Countdown zu beginnen.

Tipps für die beste Performance
Stativ: Die Kamera muss absolut ruhig stehen. Jedes Wackeln führt zu Fehlern bei der Analyse.
Beleuchtung: Achte auf gleichbleibendes Licht. Starke Schattenwürfe im Bild können die Erkennung beeinflussen.
Stromversorgung: Da die Bildanalyse CPU-intensiv ist, schließe dein Handy während des Rennens an eine Powerbank an.
App-Modus: Füge die Webseite über dein Browser-Menü zum "Startbildschirm" hinzu, um sie wie eine native App im Vollbildmodus zu nutzen.
